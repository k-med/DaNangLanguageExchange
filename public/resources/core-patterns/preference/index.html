Preference

A fast pattern for comparisons, recommendations, and personal taste.

Starter Patterns

Template

[Subject] thích [A] hơn [B]
Practice question Bạn thích [A] hay [B] hơn?

Examples

  • Tôi thích cà phê Việt Nam hơn cà phê Mỹ.
  • Tôi thích phở hơn bún bò.

Prompts

  • Bạn thích [A] hay [B] hơn?
  • Bạn thích ăn ở quán nào hơn?
  • Bạn thích học buổi sáng hay buổi tối hơn?

This pattern gives beginners a reliable way to answer with more than one word.
